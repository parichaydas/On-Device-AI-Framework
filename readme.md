Design and Implementation of an On-Device AI Framework
=====================================================

This repository contains the Design and Implementation of an On-Device AI Framework that uses Qdrant Embedded vector search for offline intelligent applications. The framework is targeted at resource-constrained devices and focuses on privacy-preserving, low-latency retrieval and on-device inference.

Contents
- Framework/: core docs, examples, source, and tools for building the on-device AI framework.

Quick start
1. See the project introduction: `Framework/INTRODUCTION.md`.
2. Read the architecture overview: `Framework/ARCHITECTURE.md`.
3. Try the example scaffold: `Framework/EXAMPLES/example_simple`.
